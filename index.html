Инструкция

  • Выберите задачу в колонке, чтобы открыть её доску внизу.
  • Импорт пространства будет отклонён, если такое пространство уже есть в списке.
  • В календаре можно выделить несколько задач через Ctrl + ЛКМ, а на телефоне открыть меню коротким зажатием по задаче.
  • Если выделено две и более задач, в меню появляется кнопка Создать группу.
  • Выделение заметки: Ctrl + ЛКМ по заметке.
  • Удаление выделенного: нажмите клавишу Delete.
